### Help on Marcdown Marcdown is a simple online Markdown editor. It allows you to type your Markdown text content and see the output instantly on the right pane. You can also download the Markdown text you wrote as a text file, Markdown file or a pdf. ### What is Markdown? Markdown is a lightweight markup language which is used to convert your text files into various other formats. Markdown is extensively used as README files as a part of repositories on github and on many other sites. ### How do I learn to write Markdown text? Writing your first Markdown text isn't tricky at all. Its possibly the easiest thing you could learn today. In fact, this help page is written in Markdown. For instance, your could write a text in bold by simply writing your content within two asteriks (\*). Thus, `**yourtext**` will look like **yourtext**. **Further reading** 1.
